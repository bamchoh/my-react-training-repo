# my-react-training-repo

This repo is for my react training.

# Update bindata

```
> npm run bindata
```

# Run webpack with watch mode

```
> npm run dev
```

# Build server.exe

```
> cd server
> go build
```

# Run server.exe

```
> server/server.exe
```
